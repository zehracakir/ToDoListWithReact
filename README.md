# React ile To Do List

- İlk olarak çalışacağımız dizinde “create-react-app” komutunu çalıştırıyoruz. React dosyalarını bu komut ile indirmiş oluyoruz.
- Daha sonra “npm start” ile projemizi çalıştırıyoruz.
